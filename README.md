# ASPI Website — Replacement Stack (Static, Multi‑page, Bilingual)

## What you do
1) Delete/replace your existing project files with the contents of this zip.
2) Keep the same folder structure.
3) Commit to GitHub and deploy (GitHub Pages / Vercel / Netlify).

## Key folders
- HTML pages: / (root)
- CSS: /assets/css/styles.css
- JS: /assets/js/main.js
- Banners: /assets/images/banners/
- Branding: /assets/images/branding/

## Mobile dropdown menu
Works automatically from main.js (hamburger + overlay + scroll lock).

## Language toggle
English/Urdu stored in localStorage (aspi_lang).
