How to run locally:
- Double click index.html, then use the English/Urdu toggle on top.

Deploy with GitHub → Vercel:
1) Create GitHub repo from this folder and push to GitHub.
2) In Vercel: Add New → Project → Import the repo
3) Framework: Other. Build/Output: leave empty.
4) Deploy.

Updates:
- Edit files, commit, push. Vercel redeploys automatically.
